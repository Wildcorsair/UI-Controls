Описание классов фреймворка UI-Controls


Общие классы

f-right Тип: Класс. Задает выравнивание блоков по правому краю. Применяется к любым блочным елементам.
f-left Тип: Класс. Задает выравнивание блоков по левому краю. Применяется к любым блочным елементам.

Классы таблиц

gray-table Тип: Класс. Задает стиль таблиц. Применяется к елементам типа <table>.
counter Тип: Класс. Задает стиль блока в котором выводится информация по количеству отображаемых записей на текущей странице и количества записей удовлетворяющих условию выборки.

Применяется к елементам типа <div>.

Используется внутри класса gray-table строка <tfoot> в ячейке таблицы <td>
pagination Тип: Класс. Задает стиль блока в котором выводится "пагинация".

Применяется к елементам типа <div>.

Используется внутри класса gray-table строка <tfoot> в ячейке таблицы <td>
r-align Тип: Класс. Задает выравнивание текста по правому краю в ячейке таблицы.

Применяется к елементам типа <td>.

Используется внутри класса gray-table в "теле" таблицы <tbody>

btn-cont Тип: Класс. Является контейнером для кнопки в ячейке таблицы (ширина контейнера 24px).

Применяется к елементам типа <td>.

Используется внутри класса gray-table в "теле" таблицы <tbody>


Классы для кнопок

btn-tb Тип: Класс. Задает стиль кнопок типа "Toolbar button". Применяется к елементам типа <button>, <a>, <input> и <div>.
